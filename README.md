# Flappy Bird

The famous game made with C++ and SFML

---

# Dependencies

+ [g++](https://github.com/gcc-mirror/gcc)
+ [SFML](https://github.com/SFML/SFML/)
+ [Make](https://github.com/wkusnierczyk/make)

---

# Building and Run

```bash
git clone https://github.com/Moneeeeee/FlappyBird.git
cd FlappyBird
make
```
